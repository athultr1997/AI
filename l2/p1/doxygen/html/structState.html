Detailed Description

Represents the state of the problem at any moment during the search.

The state at any moment is represented by:
1) State::selected_bids_from_companies
2) State::allocated_coal_blocks

◆ selected_bids_from_companies

int* State::selected_bids_from_companies

Stores the indices of the selected bids from InputData. It stores one bid from the set of bids proposed by a company. The index value corresponding to the bid data in InputData::bid_data is stored. A company from which no bid is sleceted has no_bid value set at its index. Such a definition was used for the ease of change inside the code of generate_successor.
