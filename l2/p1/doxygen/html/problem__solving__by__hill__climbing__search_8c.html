◆ take_input()

struct InputData* take_input ( char  file_name[])

Takes in input from the file name specified.

Example of input file - "1.txt". The input is stored in a 3-D array. The state is taken as the total revenue from allocating the coal blocks at that instant. The maximum value of the revenue is the sum of max revenue offered by each company since all other combinations will give revenue less than this amount. This maximum value is taken as the goal state. The solution obtained can't reach this value but will be the nearest to it.
